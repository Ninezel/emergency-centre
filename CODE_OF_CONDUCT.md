# Code of Conduct

This project is for building software communities can trust in stressful moments. That requires direct, respectful collaboration.

## Expected behavior

- Be respectful, precise, and constructive.
- Focus criticism on ideas, code, and operational outcomes.
- Assume good intent, but do not hand-wave over safety, privacy, or reliability concerns.
- Make room for contributors with different backgrounds, levels of experience, and local realities.

## Unacceptable behavior

- Harassment, intimidation, or personal attacks
- Discriminatory language or exclusionary behavior
- Deliberate misinformation about product behavior or safety characteristics
- Publishing private or sensitive data without consent

## Scope

This code of conduct applies in project discussions, issues, pull requests, and other community spaces tied to the project.

## Enforcement

Project maintainers may remove, edit, or reject contributions and participation that violate this code of conduct.
